# Dev with Mercedes | Professional Portfolio

Welcome to the official repository of **Dev with Mercedes** (Michelle Nhyira Amfo). This is a high-performance, premium digital portfolio designed to showcase a curated collection of full-stack engineering projects and technical expertise.

 

## 🚀 Overview

This portfolio has been transformed from a single-page scrolling site into a robust, **multi-page digital experience**. It emphasizes "Compact Luxury"—a design language that merges geometric silhouettes, high-density professional layouts, and smooth, cinematic motion.

### Key Pages
- **Home**: A comprehensive overview of core services, philosophy, and featured work.
- **Work**: A detailed archive of selected projects with a deep dive into engineering methodology.
- **Skills**: A technical toolkit showcasing expertise in Frontend, Backend, and Infrastructure.
- **About**: The story of Michelle Nhyira Amfo, featuring her journey, milestones, and design ethos.
- **Contact**: A professional hub for inquiries, complete with availability indicators and FAQ.

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Routing**: [React Router DOM](https://reactrouter.com/)
- **Motion**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Deployment**: Optimized for [Vercel](https://vercel.com/)

## 💎 Design Philosophy: "Effortless Elegance"

The site follows a strict design system:
- **Dark Aesthetic**: A premium `#050505` background for high contrast.
- **Typography**: A mix of elegant Serifs for headers and precise Monospaced fonts for technical data.
- **Motion**: Subtle micro-interactions and scroll-triggered animations to guide user focus.
- **Grid System**: A clean, border-heavy grid that emphasizes structural integrity.

## 📦 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/michelleamfo2007-blip/chelle-portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
   The site will be available at `http://localhost:3001` (or your local port).
